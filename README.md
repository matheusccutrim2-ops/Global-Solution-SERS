Resumo da Solução

Este projeto analisa o consumo energético de um ambiente de trabalho utilizando uma base de dados simulada de 60 dias (registros horários). A partir dessa análise, o sistema identifica padrões de uso, detecta desperdícios e simula melhorias baseadas em automação inteligente e energia renovável.

A solução combina:

Análise de dados para compreender o comportamento do consumo.

Simulação IoT para redução de energia em horários ociosos.

Simulação de energia solar para estimar compensação e economia.

O objetivo é propor uma abordagem sustentável e alinhada ao futuro do trabalho, reduzindo custos e impactos ambientais.

* Como Executar
1. Pré-requisitos

Python 3.10+

Bibliotecas:

pandas
numpy
matplotlib
seaborn

2. Execução Local

Baixe o repositório.

No terminal, execute:

python main_colab.py

* O script irá:

carregar o CSV

gerar gráficos

exibir análises e simulações

3. Execução no Google Colab

Abra o Colab.

Envie:

consumo_simulado.csv

main_colab.py

Execute todas as células.

* Resultados Obtidos

Identificação de picos de consumo no horário comercial.

Detecção de desperdício fora do expediente.

Economia estimada com automação inteligente.

Redução de até 30% do consumo diário com energia solar (cenário simulado).

* Benefícios da Solução

Redução de desperdício energético.

Menor impacto ambiental.

Economia operacional.

Ambientes de trabalho mais inteligentes e sustentáveis.
